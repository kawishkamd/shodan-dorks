# Shodan Dork Generator

Offline browser-based tool to build Shodan search queries using templates and filters.

## Features
- Visual query builder
- Prebuilt dork templates
- ASN database (Sri Lanka + global)
- Filter reference panel
- Copy query or open directly in Shodan

## Usage
Just open `shodan-dork-generator.html` in any browser. No setup needed.
